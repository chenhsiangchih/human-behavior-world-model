# Human Behavior World Model

## Vision

Build a world model that predicts human behavior in autonomous driving scenarios before dangerous actions become obvious.

## Problem

Current autonomous systems often react after pedestrians become visible.

Can we predict whether an occluded pedestrian
will enter the roadway within the next 2 seconds?

## Long-Term Goal

Predict:

- Future pedestrian trajectory
- Crossing intention
- Collision risk
- Human behavior in urban environments

## Datasets

Planned:

- PIE
- JAAD
- CARLA
